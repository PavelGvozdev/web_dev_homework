Заголовок 1

Функция jQuery (селектор, контекст) предназначена для выполнения поиска элементов в DOM-дереве на основании селектора, указанного в качестве первого аргумента. По умолчанию поиск элементов функция jQuery начинает с корня DOM-дерева (если второй аргумент не указан). Если же необходимо выполнить поиск элементов внутри определённого контекста, то его необходимо указать во втором аргументе. В качестве результата данная функция возвращает объект jQuery, содержащий найденные DOM-элементы.
